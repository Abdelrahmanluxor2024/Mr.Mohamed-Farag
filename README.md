# 📚 منصة الأستاذ محمد فراج التعليمية

> **معاً نحوَ تفوقك** - منصة تعليمية متكاملة لمادة التاريخ

## 📋 نبذة عن المنصة

منصة تعليمية احترافية للأستاذ **محمد فراج** - مدرس مادة التاريخ، تغطي المراحل من الصف الثاني الإعدادي إلى الصف الثالث الثانوي (عام وبكلوريا).

### المميزات:
- ✅ نظام امتحانات إلكتروني بأكواد سرية
- ✅ تصحيح تلقائي فوري مع الشرح
- ✅ لوحة تحكم للمدرس مع إحصائيات ورسوم بيانية
- ✅ تصميم عربي RTL احترافي
- ✅ متجاوب مع كل الأجهزة (Mobile First)
- ✅ تصدير النتائج CSV
- ✅ مؤقت تنازلي مع تسليم تلقائي

## 🚀 التثبيت والتشغيل

### 1. تثبيت المشروع
```bash
npm install
```

### 2. إعداد قاعدة البيانات (Supabase)

اذهب إلى [Supabase Dashboard](https://app.supabase.com) → SQL Editor وشغل الملفات بالترتيب:

```
sql/01_system_schema.sql      ← أولاً: إنشاء الجداول
sql/02_exam_history_grade2_prep.sql  ← امتحان ثاني إعدادي
sql/03_exam_history_grade3_prep.sql  ← امتحان ثالث إعدادي
sql/04_exam_history_grade1_sec.sql   ← امتحان أول ثانوي
sql/05_exam_history_grade2_sec.sql   ← امتحان ثاني ثانوي
sql/06_exam_history_grade3_sec.sql   ← امتحان ثالث ثانوي
```

### 3. تشغيل المشروع
```bash
npm run dev
```

### 4. بناء المشروع للإنتاج
```bash
npm run build
```

## 🔑 أكواد الامتحانات الافتراضية

| المرحلة الدراسية | كود الامتحان |
|---|---|
| الصف الثاني الإعدادي | `HIST-2PREP-001` |
| الصف الثالث الإعدادي | `HIST-3PREP-001` |
| الصف الأول الثانوي | `HIST-1SEC-001` |
| الصف الثاني الثانوي | `HIST-2SEC-001` |
| الصف الثالث الثانوي | `HIST-3SEC-001` |

> ⚠️ الأكواد سرية ويعطيها المدرس للطلاب فقط

## 👨‍💼 لوحة التحكم (الإدارة)

- **الرابط:** `/admin`
- **كلمة المرور:** `admin123`

### مميزات لوحة التحكم:
- إحصائيات سريعة (إجمالي الطلاب، متوسط الدرجات)
- جدول النتائج مع فلاتر وبحث
- تصدير النتائج بصيغة CSV
- إدارة أكواد الامتحانات (تفعيل/تعطيل)
- رسوم بيانية (متوسط الدرجات، عدد الطلاب يومياً)
